## Text Classification with Generative Models

This repository contains a Jupyter Notebook that explores text classification using generative language models.
The notebook demonstrates how generative models can be adapted for classification tasks, including data preprocessing, prompt or model setup, inference, and evaluation.

### Execution Environment
This notebook was developed and run using Google Colab.
To avoid dependency issues and ensure consistent results, it is strongly recommended to open and run this notebook in Google Colab rather than locally.
Running the notebook locally may require additional configuration and library installation.
